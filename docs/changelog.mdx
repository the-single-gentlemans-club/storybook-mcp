## [0.10.5] — 2026-02-16

### Added

- **React Native Web auto-configuration** — generates `vite.config.ts` with React Native Web aliasing for React Native/Expo projects
- **App directory story paths** — stories in `app/**/*.stories.*` included automatically for Expo Router projects

### Changed

- Setup command now detects React Native/Expo projects and configures appropriate tooling

---

## [0.10.4] — 2026-02-16

### Fixed

- **Critical:** prop extraction failing on Windows due to CRLF line endings — story files were generated with no args
- Union type detection not recognizing double-quote string literals (`"light" | "medium"`)
- Improved default args for common prop names (`title`, `label`, `placeholder`, `text`, `value`, `name`, `id`)
- Better defaults for optional boolean and number props

---

## [0.10.1] — 2026-02-16

### Fixed

- **Test suite** — downgraded Vitest 4.x → 3.2.4 to resolve "failed to find the runner" errors
- **Router detection** — added `react-router-dom` to router package detection
- **TypeScript module resolution** — changed from `bundler` to `node`
- All 124 tests passing

### Verified

- Framework-specific docs generation for Chakra, Tamagui, Gluestack, shadcn, vanilla
- Component-level framework detection via import analysis
- MSW template timeout correct at 5000ms
