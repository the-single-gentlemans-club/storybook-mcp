## What is Storybook MCP?

Storybook MCP is a **Model Context Protocol server** that automates Storybook story generation, component analysis, and validation. It integrates with Claude, Cursor, and other MCP-compatible AI assistants to supercharge your Storybook workflow.

It **auto-detects** Chakra UI, shadcn/ui, Tamagui, and Gluestack UI. It works with any React project — unrecognized frameworks use vanilla defaults.

## Why `forgekit` in the package name?

**ForgeKit** is the broader product that scaffolds Nx monorepos (app, UI library, Storybook, and related tooling). This MCP is meant to sit alongside that workflow as the **Storybook + MCP** piece. The **`@forgekit` npm scope is not available**, so the package is published as **`forgekit-storybook-mcp`** (unscoped). The CLI also exposes the shorter command **`storybook-mcp`**. This repo is **MIT and standalone** — you do not need the rest of the ForgeKit generator to use it.

## Key Features

<AccordionGroup>
  <Accordion icon="rocket" title="Auto-Sync on Startup">
    When the MCP server starts, it automatically:
    - Scans all components in configured libraries
    - Creates missing stories, tests, and MDX docs
    - Updates existing files when components have changed
    - Caches component hashes for efficient change detection
  </Accordion>

  <Accordion icon="palette" title="Framework Support">
    Built-in detection and optimization for:
    - **Chakra UI** - ThemeProvider, responsive props, color modes
    - **shadcn/ui** - Radix primitives, CVA variants
    - **Tamagui** - Native + web styling, responsive design
    - **Gluestack UI** - Component composition patterns
    - **React Native** - Platform-specific rendering
    - **Vanilla React** - Universal fallback for any framework
  </Accordion>

  <Accordion icon="code" title="15 Powerful Tools">
    - Component listing and filtering
    - Deep component analysis
    - Story generation with templates
    - Smart story updates (preserves custom exports)
    - Test generation (Playwright/Vitest)
    - MDX documentation generation
    - Figma Code Connect generation
    - Story validation
    - Coverage tracking
    - Health checks and diagnostics
  </Accordion>

  <Accordion icon="sparkles" title="8 Story Templates">
    - Basic - Quick documentation
    - With Variants - Showcase all sizes/variants
    - With MSW - API mocking
    - With Router - Navigation components
    - Interactive - Play functions
    - Form - Validation and submission
    - Page - Full-page layouts
    - Responsive - Multi-viewport testing
  </Accordion>
</AccordionGroup>
